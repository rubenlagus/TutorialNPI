Detailed Description

This class (extended from Activity) implements a Touch listener above the view in order to determine when the screen is touched and how many finger are there on it.

Author
Ruben Bermudez
Santiago Tapia
Isaac Morely
Note
Even this application support up to 10 fingers, this number depend on the device. Moreover, one finger that touch on 2 different moment the screen can receive a diferent ID each time.
